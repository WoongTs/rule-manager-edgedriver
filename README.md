# AEB RuleGen Edge Driver

SmartThings Edge Driver package for creating and managing SmartThings Rules through AEB/EdgeBridge.

## Package

```powershell
smartthings edge:drivers:package .\driver
```

## Notes

- Configure `Bridge Address` as `IP:port`, or leave it empty for EdgeBridge mDNS discovery.
- Keep PATs, tokens, account IDs, location IDs, device IDs, hub IDs, and channel IDs out of commits.
